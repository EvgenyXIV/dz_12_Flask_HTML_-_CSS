ДЗ_12 HTML & CSS

Базовое веб-приложение на Flask.


HTML и CSS — это основные инструменты веб-разработчика, которые используются для создания сайтов и веб-приложений. Изучение HTML и CSS необходимо для понимания основ веб-разработки, создания собственных проектов, повышения квалификации, развития навыков логического мышления и знакомства с основами программирования. В нашем случае изучение HTML и CSS позволит создавать красивые и правильно функционирующие сайты
